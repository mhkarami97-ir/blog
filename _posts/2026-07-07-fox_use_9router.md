## معرفی و کاربرد اصلی
9Router یک Gateway محلی (روی سیستم خودتان اجرا می‌شود) است که یک endpoint واحد و سازگار با فرمت OpenAI ارائه می‌دهد و همه ابزارهای CLI/IDE هوش مصنوعی شما را به بیش از ۶۰ پروایدر (Claude، GPT، Gemini، DeepSeek، GLM و...) وصل می‌کند. این پروژه در GitHub بیش از ۱۵ هزار ستاره دارد و به‌صورت فعال توسعه می‌یابد.   
مشکلی که حل می‌کند این است که سهمیه اشتراک‌های ماهانه (مثل Claude Pro یا Copilot) معمولاً بدون استفاده منقضی می‌شود و Rate Limit وسط کار مانع ادامه کدنویسی می‌شود؛ 9Router با روتینگ سه‌لایه (اشتراک → ارزان → رایگان) این مشکل را رفع می‌کند. 

## نصب و راه‌اندازی اولیه
نصب فقط با دو دستور ساده npm انجام می‌شود:

```
npm install -g 9router
9router
```

پس از اجرا، داشبورد به‌صورت خودکار در آدرس `http://localhost:20128` باز می‌شود. برای اجرای از طریق Docker هم می‌توانید از ایمیج رسمی روی Docker Hub یا GHCR استفاده کنید: 

```
docker run -d --name 9router -p 20128:20128 \
  -v "$HOME/.9router:/app/data" \
  -e DATA_DIR=/app/data \
  decolua/9router:latest
```

## اتصال پروایدرهای هوش مصنوعی
پروایدرها در سه دسته تقسیم می‌شوند و هرکدام روش اتصال متفاوتی دارند: 

| دسته | نمونه پروایدر | روش اتصال | هزینه |
|---|---|---|---|
| اشتراکی (OAuth) | Claude Code, Codex, GitHub Copilot, Cursor | لاگین OAuth از داشبورد | ماهانه ۱۰ تا ۲۰۰ دلار   |
| ارزان (API Key) | GLM, MiniMax, Kimi | ثبت API Key در داشبورد | هر ۱ میلیون توکن ۰.۲ تا ۰.۶ دلار   |
| رایگان | Kiro AI, OpenCode Free, Vertex AI | OAuth یا بدون نیاز به احراز هویت | صفر تا ۳۰۰ دلار اعتبار هدیه   |

برای شروع سریع و بدون هزینه، ساده‌ترین راه اتصال به Kiro AI (کلود ۴.۵ نامحدود رایگان) یا OpenCode Free است که نیاز به هیچ ثبت‌نامی ندارد. 

## ساخت Combo برای Fallback خودکار
ویژگی کلیدی 9Router، «Combo» است؛ یعنی زنجیره‌ای از چند مدل که وقتی سهمیه اولی تمام شود، به‌طور خودکار به بعدی سوییچ می‌کند:   
این ساختار در بخش Combos داشبورد ساخته می‌شود و می‌توانید نام دلخواه به آن بدهید تا در ابزارهای CLI استفاده کنید. 

## اتصال به ابزارهای کدنویسی
هر ابزار روش تنظیم مخصوص خود را دارد؛ نکته مشترک همه، آدرس‌دهی به `http://localhost:20128/v1` است:   
- Cursor IDE: در تنظیمات Models، آدرس OpenAI API Base URL را روی `localhost:20128/v1` قرار دهید و API Key دریافتی از داشبورد را وارد کنید. 
- Claude Code: فایل `~/.claude/config.json` را ویرایش کرده و `anthropic_api_base` را به همان آدرس تغییر دهید. 
- Codex CLI: با دستورات `export OPENAI_BASE_URL` و `export OPENAI_API_KEY` در ترمینال تنظیم می‌شود. 
- Cline / Continue / RooCode: پروایدر را روی «OpenAI Compatible» بگذارید و همان Base URL و کلید را وارد کنید. 

## ویژگی‌های صرفه‌جویی در توکن
دو قابلیت به‌صورت پیش‌فرض فعال هستند که مصرف توکن را کاهش می‌دهند:   
- RTK Token Saver: خروجی دستوراتی مثل git diff، grep یا ls را قبل از رسیدن به مدل، به‌صورت بی‌ضرر فشرده می‌کند و بین ۲۰ تا ۴۰ درصد در توکن ورودی صرفه‌جویی می‌کند.
- Caveman Mode: با تزریق یک پرامپت سیستمی خاص، پاسخ مدل را فنی اما کوتاه‌تر می‌کند و تا ۶۵ درصد در توکن خروجی صرفه‌جویی ایجاد می‌کند.

## نکته مهم درباره هزینه‌ها
مطابق مستندات رسمی، خود 9Router هرگز هزینه‌ای از کاربر نمی‌گیرد و عدد «Cost» نمایش داده‌شده در داشبورد صرفاً یک شاخص مقایسه‌ای برای نشان دادن میزان صرفه‌جویی است، نه صورت‌حساب واقعی. پرداخت واقعی فقط در صورت استفاده از پروایدرهای اشتراکی یا ارزان و مستقیماً به همان شرکت انجام می‌شود. 

## استقرار روی سرور و دیپلوی
برای استفاده روی VPS، پروژه را کلون و با متغیرهای محیطی زیر اجرا می‌کنید: 

```
git clone https://github.com/decolua/9router.git
cd 9router
npm install && npm run build
export JWT_SECRET="your-secure-secret"
export PORT="20128"
npm run start
```
